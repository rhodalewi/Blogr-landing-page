# Blogr landing page solution

A fully responsive and interactive landing page built as part of a Frontend Mentor challenge. This project focuses on modern UI implementation, reponsive design principles and production-style navigation behaviour including hover-based dropdown systems and mobile friendly navigation.

## Overview
This Blogr landing page is a modern marketing website. It features:
- Responsive navigation menu
- Dropdown menus on hover for desktop
- Smooth arrow icon rotation
- mobile friendly layout with hamburger menu
- Responsive layout for all screen size
